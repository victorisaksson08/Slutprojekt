## Retro Car Game

A retro-style car dodging game built with C# Windows Forms in Visual Studio 2022.

### How to Play
- Use **Left/Right Arrow** keys or **A/D** to switch lanes
- Dodge oncoming traffic to score points
- Speed increases every 5 points scored
- Try to beat your high score!

### Features
- 5-lane road with retro pixel-art style graphics
- Increasing difficulty as your score goes up
- High score tracking (per session)
- Smooth 60 FPS gameplay
- Retro color palette and visual design

### How to Run
1. Open `RetroCarGame.sln` in Visual Studio 2022
2. Press **F5** to build and run
3. Press **Enter** or **Space** to start the game

### Requirements
- Visual Studio 2022
- .NET 6.0 or later (Windows)
